# Prompts Used - ShopZone (Week 6)

## Project Goal Prompt
Build a multi-page e-commerce SPA using React Router, Context API, dynamic product routing, cart management, and protected checkout.

## Implementation Prompts
- Set up routes: `/`, `/shop`, `/contact`, `/product/:id`, `/cart`, `/login`, `/checkout`
- Fetch products from `https://dummyjson.com/products`
- Add dynamic product details using `useParams`
- Build global cart context with add/increase/decrease/remove
- Show live cart badge in navbar
- Persist cart with localStorage
- Create fake auth (`Login as Guest`) and protected checkout route
- Add Vercel SPA rewrite config via `vercel.json`

## UI/Design Prompts
- Convert default layout into Amazon-inspired e-commerce design
- Improve product cards, filters sidebar, detail buy box, and cart summary
- Fix responsive CSS overlap issues between filters and product grid

## Deployment Prompts
- Push source to GitHub repository
- Deploy to Vercel
- Verify route refresh behavior and production URLs
